# Power-ups

- Certain blocks will contain powerups
- these blocks will have a special effect on them to show the powerup

## Ball damage type modifiers

### Fire

- can destroy blocks that are weak to fire without bouncing off them

### Ice

- can freeze blocks upon collision in a certain radius
- frozen blocks can be broken without bouncing

### Explosive

- can break tougher blocks
- on collision, will damage nearby blocks

### Magnetic

- ball will drift towards the closest block with certain types


## Ball property modifiers

### Multiplier

- Will split the ball into multiple balls

### Speed modifier

- will speed up/slow down the ball

### Size modifier

- Will change the ball size


## Paddle Weapons(?)

- affects the gameplay in a certain way

### Laser

- Can shoot lasers/projectiles to destroy blocks

### Telekinesis(?)

- the balls x position will drift towards the paddles x position

### Bomb

- similar to laser, a single projectile with AOE damage
  

## Paddle modifiers

### Size

- Will change the paddle's size

### Magnet

- Will catch the ball instead of bouncing it
  

## Global Powerups

- can be used anytime once (wip) during a screen

### Extra powerups

- will put powerups into more blocks

### Nuke

- will destroy blocks

### Block Weakener - placeholder name

- will weaken tough blocks
- unbreakable blocks become breakable
- multi-layered blocks will lose layers

### XP booster

- this activates globally and has global timer
- will give extra xp collected while active
